# 実験要約：救急搬送における客観的乗り心地評価モデルの構築

## 1. 企画の経緯（背景）
救急自動車の運転技術評価は、これまで熟練隊員の「主観的な感覚」や「経験則」に大きく依存してきました。しかし、搬送される傷病者にとっては、単一の揺れだけでなく、加速・減速・旋回などが複雑に組み合わさった揺れが大きな苦痛（不快感）につながります。若手隊員への教育や安全な搬送の質を担保するため、主観に頼らない「客観的かつ定量的な評価指標」の確立が求められていました。

## 2. 実験の目的
本実験の主な目的は以下の通りです。
- **不快感の予測モデル構築**: スマートフォンのセンサーを活用し、機械学習アルゴリズムを用いて「傷病者がいつ不快と感じるか」を高精度に予測するモデルを構築する。
- **要因の可視化**: SHAP（機械学習の説明性手法）等を用いて、不快感に影響を与える具体的な運転操作や振動特性（G、躍度など）を特定する。
- **教育・フィードバックへの応用**: 抽出された客観的数値を基に、若手機関員の運転技術向上を支援するフィードバックシステムの基礎を作る。

## 3. 実験方法
- **データ収集**: 専用開発したスマートフォンアプリを用い、加速度（G）、躍度（Jerk）、ヨーレート、速度、位置情報を50Hz（毎秒50回）で記録。
- **被験者設定**: 視覚や聴覚による予測を排除するため、アイマスクと耳栓を着用した被験者をストレッチャーに固定し、実際の市街地等を走行。
- **アノテーション**: 被験者が「不快」と感じた瞬間に手元のフラグを入力し、走行データと同期。
- **解析手法**: 
  - 勾配ブースティング決定木（LightGBM）を用いた二値分類。
  - Optunaによるハイパーパラメータの自動最適化。
  - 5シードのアンサンブル学習による推論の安定化。
  - SHAP値による特徴量寄与度の分析。

## 4. 途中経過（解析結果）
現在までの解析で以下の成果が得られています。
- **極めて高い予測精度**: 機械学習モデルは **AUC 0.9966** という極めて高い精度で不快感を識別可能であることを実証。
- **支配的な要因の特定**: 
  - 不快感には「単なるGの大きさ」以上に、**「垂直方向の躍度の乱れ（Jerk_Zの分散）」**が最も強く寄与していることが判明。
  - 運転者の属性（年齢・経験）も上位因子となり、運転スタイルが乗り心地に直結することが数値で示された。
- **数値的指標の確立**: 
  - 快適な搬送のための具体的な閾値を算出。例えば、前後・左右ともに **0.1G を超える突発的な揺れを抑制すること**が一つのガイドラインとして抽出された。
- **汎用性の確認**: 専用の計測機器を必要とせず、スマートフォンのみでこれらの高度な分析が可能であることを確認。
