## 2. 精度向上のための学習設定

Google Colabの強力な計算リソースを活用し、以下の通り「学習の徹底度」を引き上げます。

- **ハイパーパラメータ探索 (Optuna):** 20回 → **100回** に増加。より緻密なパラメータの組み合わせ（学習率、木の深さ等）を追求します。
- **アンサンブル (Seed Bagging):** 5つ → **10回** に増加。10個の異なる視点を持つモデルを平均化することで、一部の特異なデータに惑わされない「安定感のある予測」を実現します。
- **損失関数の高度化:** 通常の `binary_logloss` から **Focal Loss** へ変更。これにより、データの大半を占める「何事もない平穏な走行」を学習しすぎるのを防ぎ、稀に発生する「不快な瞬間」の学習に重みを置きます。
